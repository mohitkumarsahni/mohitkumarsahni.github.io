# Mohit Kumar Sahni - Personal Profile

A minimal personal introduction page hosted with GitHub Pages.

Edit `index.html` to update the introduction, tech stack, and contact links. Visual styles live in `styles.css`.

## Preview locally

Open `index.html` directly in a browser, or run any static file server from this directory.

## Publish

Push the files to the `main` branch. In the repository's GitHub settings, configure Pages to deploy from the root of `main`.
